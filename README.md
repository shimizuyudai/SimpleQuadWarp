# SimpleQuadWarp

This tool can be used to deform multiple planes.

* Video
* Image
* Spout

# KeyAssign
| Key | Action |
|:-|:-|
| Return       | Save Settings        |
| C      | Clear coordinate editing        |
| R       | Release the editing quad        |


# Dependency

* OpenCVForUnity
* Json.NET
* [UtilPack4Unity](https://github.com/shimizuyudai/UtilPack4Unity)
